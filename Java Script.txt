# 자바스크립트란?

웹에서 쓰이는 *하나뿐인* 프로그래밍 언어!
=> 웹은 interactive하게 만드는 일을 한다.

웹이 발전할수록 JavaScript가 할 수 있는 일은 무궁무진하게 많아질것이다! 
=> 즉 전망이 매우 좋다!

JavaScript를 사용하면 게임, 인터렉티브 사이트, 실시간 채팅 등 다양한 것을 만들 수 있다.

# 바닐라 자바스크립트?

라이브러리가 없는 자바스크립트이다.
그저 브라우저가 제공하는 기본 자바스크립트이다.

바닐라 자바스크립트를 하고 넘어가야한다. 
JavaScript에 재능이 있는 사람은 찾기 힘들다.
(JQuery, 리액트에 재능이 있는 사람은 찾기 쉬움..)
즉 라이브러리나 프레임웤을 다루는 사람은 찾기 쉽지만,
그 반대의 경우는 찾기가 굉장히 어렵다.

기구를 아무리 잘 다뤄도 핵심을 모른다면 말짱꽝이다!!
그 핵심이 바로 바닐라 자바스크립트라고 할 수 있다.


# let, const, var

* let
=> block-scope
let을 사용하면 var과 다르게 변수 재선언이 불가능하다.

*const(=constant)
=> 절대 바뀌지 않는 값 (상수)

*var
=> function-scope
3년 전에는 이 변수밖에 없었다.
모든 변수의 조상이라고 볼 수 있다.
